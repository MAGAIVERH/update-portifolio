Set/2023 - Jan/2024

Barber Shop Manager

Desenvolvi o Barber Shop Manager para simplificar o agendamento de serviços em barbearias. Utilizei React.js, Tailwind CSS e TypeScript no frontend e Next.js no backend, com PostgreSQL para o banco de dados. Minha Atuação: No frontend, criei uma interface intuitiva e responsiva, garantindo uma experiência fluida para os usuários. No backend, implementei uma aplicação SSR para melhorar o desempenho e integrei um banco de dados para gerenciar clientes e agendamentos. Desafios e Conquistas: Enfrentei desafios diversos, incluindo otimização de desempenho e criação de um sistema de notificações em tempo real. Essas soluções não só melhoraram a comunicação com os clientes, mas também a eficiência operacional das barbearias. Lições Aprendidas: Além do crescimento técnico, aprendi lições valiosas sobre empatia e colaboração. Este projeto reforçou meu compromisso em criar soluções tecnológicas impactantes e acessíveis.
